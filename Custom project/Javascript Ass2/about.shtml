Website and Page Structure

This website is structured into seperate content styles. There are four major sections; header, navigation links, main content and footer. These are styled using the same ID across all of the pages for a consistent content layout. The header has title and other graphics. Navigation links are a horizontally styled list that provide the structure for navigating between content pages. Main content includes all subject matter information and the footer has document control information e.g. references, links to the 'About' and 'Report' pages and links to W3C for valid XHTML and CSS certification. Content pages via the navigation links are the index page, form demonstration page and a content page to demonstrate other HTML and JavaScript features.

Each ID is self-contained so every content page has exactly the same IDs and can be interchangeably copied and pasted between each page (e.g. the footer div). Additionally because the site only uses IDs, a div ID is only usable once per page so there is no opening and closing of the same div (which is possible using classes) that can make content structuring look messy and hard to follow.
